# TagBuy 서비스 기획서

**버전**: 1.0
**작성일**: 2025년 12월 15일
**핵심**: PG + 에스크로 보유 기반 차세대 인플루언서 마케팅 플랫폼

---

## 서비스 개요

### 한 줄 정의
> **"에스크로 0.5% + PG 실비, 10분 만에 정산받는 인플루언서 마케팅 플랫폼"**

### 핵심 가치 제안

**광고주에게**
- 에스크로 수수료 0.5% (경쟁사 3% 대비 85% 절감) + PG 실비
- PG 수수료 부담 주체 선택 가능 (광고주 또는 인플루언서)
- 가상계좌 사용 시 총 비용 0.8-1% (카드 대비 최대 70% 절감)
- 투명한 에스크로 시스템으로 안전한 거래
- 원클릭 결제로 5초 만에 캠페인 시작

**인플루언서에게**
- 실시간 정산 (승인 후 10분 내 입금)
- 24시간 365일 언제든 출금 가능
- 광고주가 PG 수수료 부담 선택 시 100% 수령

**우리의 차별점**
- PG + 전자대금예치업 자체 보유
- 외부 PG 의존 없음 → 원가 절감 + 자유로운 UX
- 금융권 수준 보안과 컴플라이언스

## 주요 기능

### 1. 광고주 기능

#### 1.1 캠페인 생성

캠페인 만들기:
- 목표 설정 (브랜드 인지도, 판매 전환 등)
- 예산 입력 (최소 50만원부터)
- 타겟 인플루언서 조건
  - 팔로워 수 범위
  - 카테고리 (뷰티, 패션, 푸드 등)
  - 지역 (선택 사항)
- 제품/서비스 정보
- 가이드라인 업로드

**결제 프로세스**
1. 캠페인 예산 입력
2. PG 수수료 부담 주체 선택
   - 광고주가 PG 수수료 부담 (인플루언서 100% 수령)
   - 인플루언서가 PG 수수료 부담 (광고주 비용 절감)
3. 결제 수단 선택
   - 카드 결제: PG 2.5-3%
   - 가상계좌: PG 0.3-0.5% (추천)
4. 원클릭 결제 (저장된 결제수단)
5. 자동 에스크로 예치 (수수료 0.5%)
6. 캠페인 활성화

**특징**
- 외부 PG 리다이렉트 없음
- 결제 + 에스크로 자동 처리
- 5초 완료

#### 1.2 인플루언서 매칭

**AI 자동 매칭**
- 캠페인 목표에 최적화된 인플루언서 추천
- 과거 성과 데이터 기반 순위
- 가짜 팔로워 자동 필터링

**수동 검색 필터**
- 팔로워 수 (1K-10K, 10K-100K, 100K+)
- 카테고리
- 인게이지먼트율 (최소 기준 설정)
- 지역
- 과거 캠페인 완료율

#### 1.3 캠페인 관리

**실시간 대시보드 성과 지표**
- 도달률 (Reach)
- 인게이지먼트 (좋아요, 댓글, 공유)
- 클릭률 (CTR)
- 전환율 (구매 발생 시)
- CPE (Cost Per Engagement)

**자동 모니터링**
- 포스팅 여부 자동 확인
- 삭제/비공개 전환 시 즉시 알림
- 가이드라인 준수 여부 AI 검증

#### 1.4 콘텐츠 검수 (AI 리포트 기반)

**승인 프로세스**
1. 인플루언서가 콘텐츠 업로드 → AI가 즉시 검증 리포트 생성 (30초)
2. 광고주 검수 (24-48시간 내)
   - 콘텐츠 미리보기
   - AI 검증 리포트 (가이드라인 준수도, 해시태그, 필수 문구 등)
   - 승인 / 수정 요청 / 거절 선택
3. 승인 → 자동 정산 트리거 (D+7), 에스크로에서 자동 집행
4. 수정 요청 → 인플루언서에게 피드백, 재업로드
5. 거절 → 캠페인 취소, 에스크로 자동 환불

**AI 리포트 활용**
- 광고주는 AI 리포트를 참고하여 빠른 검수
- 객관적 기준으로 승인/거절 판단
- 분쟁 시 AI 리포트가 증거 자료

### 2. 인플루언서 기능

#### 2.1 캠페인 신청

**캠페인 탐색 정렬 옵션**
- 최신순
- 높은 보상순
- 마감 임박순
- 매칭률 높은 순 (AI 추천)

**신청 프로세스**
1. 관심 캠페인 선택
2. 간단한 지원 메시지 작성
3. 포트폴리오 첨부 (선택)
4. 신청 완료

**자동 승인 시스템**
- 과거 완료율 90% 이상 → 즉시 승인
- 신규 인플루언서 → 광고주 검토

#### 2.2 콘텐츠 제작 및 업로드

**업로드 프로세스**
1. 콘텐츠 제작
2. 인스타그램/블로그 게시
3. tagbuy 플랫폼에 URL 제출
4. 스크린샷 자동 캡처 (증빙)

**AI 자동 검증 & 리포트**
1. 콘텐츠 제출 즉시 AI 분석 (30초 내) → 자동 검증 리포트 생성
2. 검증 리포트 내용
   - 가이드라인 준수도
   - 체크리스트 (해시태그, 브랜드명 언급, 제품 이미지, 필수 문구)
   - 예상 성과 (예상 도달, 예상 인게이지먼트)
3. 리포트 공유
   - 인플루언서: 즉시 확인, 문제 발견 시 재업로드 가능
   - 광고주: 검수 시 AI 리포트 참고
4. 피드백 & 이의 제기
   - 인플루언서: "AI 판정 이의 제기" 버튼
   - 광고주: AI 리포트에 코멘트 추가
   - 양측 의견 모두 저장 (분쟁 대비)

#### 2.3 실시간 정산

**정산 옵션**
- 표준 정산 (무료): 광고주 승인 후 24시간 이내, 자동 입금
- 즉시 정산 (+0.5% 수수료): 광고주 승인 후 10분 이내, 365일 24시간 가능
- 예약 정산 (무료): 원하는 날짜/시간 지정

**출금**
- 최소 출금액: 1만원
- 출금 수수료: 무료
- 계좌 등록 (1회)
- 클릭 한 번으로 즉시 출금

#### 2.4 수익 관리

**대시보드 수익 현황**
- 이번 달 수익
- 진행 중 캠페인 예상 수익
- 정산 대기 금액
- 출금 가능 금액
- 월별 수익 그래프

**세금 관리**
- 원천징수 자동 처리 (3.3%)
- 월별 수익 리포트
- 연말 종합소득세 신고 자료 제공

### 3. 자동화 기능

#### 3.1 에스크로 자동 관리

**에스크로 집행 시스템 예시**
- 500만원 캠페인, 인플루언서 10명 선정
- 광고주 결제: 525만원 (에스크로 0.5% 포함)
- 에스크로 예치: 500만원
- 인플루언서당 예정 금액: 50만원 × 10명

**자동 집행 플로우**
1. 광고주 결제 (D-Day): 전체 예산 에스크로 자동 예치, 캠페인 활성화
2. 인플루언서 선정 & 콘텐츠 제작: 광고주가 신청자 중 선정, 금액 할당
3. 콘텐츠 게시 (D+0): URL 제출, AI 자동 검증, 광고주 검수
4. 콘텐츠 유지 확인 기간: D+1~D+7 매일 자동 크롤링, 삭제/비공개 체크
5. 자동 정산 트리거 (D+7): 조건 충족 시 에스크로에서 자동 출금, 10분 내 입금
6. 분쟁 발생 시: 양측 증거 제출, AI 1차 판단, 필요 시 인간 중재자 개입

**집행 규칙 옵션 (광고주 선택)**

| 규칙 | 정산 시점 | 적용 케이스 |
|------|----------|------------|
| 표준 (D+7) | 게시 후 7일 뒤 | 일반 캠페인 (추천) |
| 단기 (D+3) | 게시 후 3일 뒤 | 이벤트성 캠페인 |
| 장기 (D+30) | 게시 후 30일 뒤 | 장기 유지 필수 캠페인 |
| 즉시 (D+1) | 게시 후 1일 뒤 | 신속 정산 (+0.3% 수수료) |

**자동 모니터링 (매일 자동 크롤링)**
- 콘텐츠 존재 여부 확인
- 공개/비공개 상태 체크
- 삭제 감지 시 즉시 알림
- D+7 이전 삭제 시 정산 취소

**보호 장치**
- 콘텐츠 미제작 시 자동 환불 (D+14)
- 중도 삭제 시 정산 취소 및 위약금 (캠페인별 설정)
- 가이드라인 위반 시 부분 환불 (광고주 판단)
- 중간 정산 옵션 (50% 선지급 가능, D+0)

#### 3.2 AI 사기 탐지

**자동 검증 항목**
- 가짜 팔로워 비율 (봇 계정 탐지)
- 급격한 팔로워 증가 패턴
- 인게이지먼트 진정성 (댓글 품질, 좋아요 타이밍, 리치 대비 비율)
- 콘텐츠 품질 (가이드라인 준수, 금지어 사용, 이미지/동영상 품질)

**조치**
- 의심 계정: 경고 표시
- 확인된 부정: 계정 정지
- 광고주에게 즉시 알림

#### 3.3 성과 자동 추적

**자동 수집 데이터**
- 노출수 (Impressions)
- 도달률 (Reach)
- 좋아요, 댓글, 저장, 공유
- 프로필 방문
- 웹사이트 클릭 (UTM 자동 생성)

**리포트 자동 생성**
- 일일 성과 요약 (이메일)
- 주간 성과 리포트 (PDF)
- 캠페인 종료 후 최종 리포트

## 차별화 포인트

### 1. 투명한 수수료 구조

**경쟁사 (올인 3%)**
- 광고비 1,000만원 → 광고주 지급 1,030만원
- 플랫폼 수수료 30만원 (내역 불명확)

**TagBuy (에스크로 0.5% + PG 실비)**

가상계좌 선택 시:
- 광고비 1,000만원 → 광고주 지급 1,008만원
- 에스크로 수수료: 5만원 (0.5%)
- PG 수수료: 3만원 (0.3%)
- 22만원 절감 (73% 저렴)

카드 결제 시:
- 광고비 1,000만원 → 광고주 지급 1,055만원
- 에스크로 수수료: 5만원 (0.5%)
- PG 수수료: 25만원 (2.5%, 광고주 선택 시)

**핵심 차별점**
- PG 수수료 부담 주체를 광고주가 선택 가능
- 가상계좌 사용 시 경쟁사 대비 73% 저렴
- 완전 투명한 비용 구조

### 2. 실시간 정산 (업계 유일)
- 경쟁사: 1-2주 소요
- tagbuy: 승인 후 10분 내 입금
- 효과: 인플루언서 플랫폼 충성도 극대화

### 3. 원클릭 통합 UX
- 경쟁사: 외부 PG 리다이렉트
- tagbuy: 5초 완료
- 효과: 결제 이탈률 최소화

### 4. 금융권 보안
- PCI-DSS Level 1 인증
- 24/7 보안 모니터링
- 금융위원회 감독

## 수수료 구조 상세

### 기본 원칙
총 비용 = 에스크로 수수료 (0.5%) + PG 수수료 (실비)

### 결제 수단별 비용

| 결제 수단 | PG 수수료 | 에스크로 | 총 비용 | 경쟁사 대비 |
|----------|----------|---------|---------|------------|
| 가상계좌 | 0.3-0.5% | 0.5% | **0.8-1%** | **73% 절감** |
| 카드 결제 | 2.5-3% | 0.5% | **3-3.5%** | 비슷 또는 약간 비쌈 |

### PG 수수료 부담 주체 선택

광고주가 캠페인 생성 시 선택:

**옵션 1: 광고주가 PG 수수료 부담**
- 광고비 1,000만원 계약 시:
- 광고주 총 지급: 1,008만원 (가상계좌) 또는 1,055만원 (카드)
- 인플루언서 수령: 1,000만원 (100% 수령)

**옵션 2: 인플루언서가 PG 수수료 부담**
- 광고비 1,000만원 계약 시:
- 광고주 총 지급: 1,005만원 (어떤 결제 수단이든 동일)
- 인플루언서 수령:
  - 가상계좌: 997만원 (PG 0.3% 차감)
  - 카드: 975만원 (PG 2.5% 차감)

### 수수료 구조의 장점

1. **투명성**: 경쟁사는 "3% 올인"으로 내역 불투명, TagBuy는 완전 투명
2. **선택권**: 광고주가 인플루언서에게 100% 지급할지 선택 가능
3. **가상계좌 유도**: 가상계좌 사용 시 압도적으로 저렴 (0.8%)
4. **경쟁 우위**: 자체 PG 보유로만 가능한 구조, 경쟁사 모방 불가

## 타겟 고객

### 주요 타겟 (Primary)

**광고주**
- 1순위: 중소기업 (연매출 10억-100억)
  - 특징: 마케팅 예산 제한적
  - 니즈: 저렴한 수수료, 명확한 ROI
- 2순위: 스타트업
  - 특징: 빠른 성장 추구
  - 니즈: 효율적인 마케팅, 데이터
- 3순위: 마케팅 대행사
  - 특징: 여러 클라이언트 관리
  - 니즈: 관리 효율화, 낮은 수수료

**인플루언서**
- 1순위: 마이크로 인플루언서 (1만-10만 팔로워)
  - 특징: 높은 인게이지먼트
  - 니즈: 빠른 정산, 투명한 거래
- 2순위: 나노 인플루언서 (1천-1만 팔로워)
  - 특징: 니치 카테고리 강점
  - 니즈: 접근성, 간편한 사용
- 3순위: 매크로 인플루언서 (10만+ 팔로워)
  - 특징: 넓은 도달률
  - 니즈: 빠른 정산

### 부차 타겟 (Secondary)

## 사용자 플로우

### 광고주 플로우

1. **회원가입**: 이메일 인증 → 기업 정보 입력 → 결제수단 등록
2. **캠페인 생성**: 캠페인 정보 입력 → 예산 설정 → 타겟 조건 설정 → 가이드라인 작성
3. **결제 (5초)**: 원클릭 결제 → 자동 에스크로 예치
4. **인플루언서 매칭**: AI 추천 확인 → 수동 검색 → 신청 승인
5. **캠페인 진행**: 실시간 모니터링 → 콘텐츠 검수 → 승인/거절
6. **정산 완료**: 자동 정산 트리거 → 에스크로 출금 → 최종 리포트 확인

### 인플루언서 플로우

1. **회원가입**: 이메일 인증 → 인스타그램 연동 → 프로필 작성
2. **캠페인 탐색**: 매칭률 높은 순 정렬 → 관심 캠페인 필터링
3. **신청**: 지원 메시지 작성 → 포트폴리오 첨부 → 신청 완료
4. **승인 대기**: 알림 수신 → 승인 확인
5. **콘텐츠 제작**: 가이드라인 확인 → 콘텐츠 제작/게시 → URL 제출
6. **정산 (10분)**: 광고주 승인 대기 → 승인 즉시 정산 → 출금 신청 → 10분 내 입금

## 성공 요인

### 1. 자체 PG + 전자대금예치업 보유 (핵심)
- 에스크로 수수료 0.5% 실현 가능
- PG 수수료 실비 정산으로 완전 투명성
- 실시간 정산 구현 가능
- 완벽한 UX 통합 가능
- PG 수수료 부담 주체 선택 기능 (경쟁사 불가)
- **결론**: 경쟁사가 절대 모방 불가

### 2. 실시간 정산 (킬러 기능)
- 인플루언서에게 혁신적 가치
- 플랫폼 락인 효과
- 입소문 마케팅 강력

### 3. 간결한 비즈니스 모델
- 금융 서비스 없이 단순화
- 빠른 실행 가능
- 리스크 최소화

### 4. 명확한 타겟
- 중소기업 + 마이크로 인플루언서
- 언더독 시장 공략
- 빠른 성장 가능
