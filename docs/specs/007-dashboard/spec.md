# Spec: 007-dashboard

## Feature Name
성과 대시보드

## User Value (고객 가치)
- 실시간 캠페인 성과 지표 확인
- ROI 분석으로 마케팅 효율 측정
- 자동 리포트 생성으로 보고 시간 절약
- 인플루언서별 성과 비교 분석

## Target User (누가 쓸까?)
- Primary: 광고주 (캠페인 성과 분석)
- Secondary: 인플루언서 (개인 성과 확인)

## User Journey & Screen Flow

### 광고주 플로우

#### 1. 메인 대시보드
- **UI Elements**:
  - 총 광고비, 진행 중 캠페인 수, 완료 캠페인 수
  - 평균 ROI
  - 이번 달 성과 요약 (도달, 인게이지먼트, 클릭)
  - 진행 중 캠페인 카드
- **CTA**: "새 캠페인 만들기", "캠페인 상세보기"
- **Next**: 캠페인별 대시보드

#### 2. 캠페인별 대시보드
- **UI Elements**:
  - 성과 지표
    - 총 도달 (Reach)
    - 총 인게이지먼트 (좋아요, 댓글, 공유, 저장)
    - 클릭률 (CTR)
    - 전환율 (연동 시)
    - CPE (Cost Per Engagement)
    - ROI
  - 일별 추이 그래프
  - 인플루언서별 성과 비교
- **CTA**: "리포트 다운로드", "기간 변경"
- **Next**: 리포트 생성

#### 3. 인플루언서별 성과
- **UI Elements**:
  - 인플루언서 목록 (도달, 인게이지먼트, CPE 순위)
  - 개별 콘텐츠 성과
  - 예상 vs 실제 성과 비교
- **CTA**: "상세 보기", "재계약 제안"
- **Next**: 인플루언서 상세

#### 4. 리포트 생성/다운로드
- **UI Elements**:
  - 리포트 유형 선택 (일간/주간/캠페인 종료)
  - 포함 항목 선택
  - 포맷 선택 (PDF/Excel)
- **CTA**: "리포트 생성", "자동 발송 설정"
- **Next**: 리포트 다운로드

### 인플루언서 플로우

#### 1. 내 성과 대시보드
- **UI Elements**:
  - 총 콘텐츠 수, 평균 인게이지먼트율
  - 월별 수익 추이
  - 카테고리별 성과
  - 최근 콘텐츠 성과
- **CTA**: "캠페인 찾기", "프로필 업데이트"
- **Next**: 캠페인 탐색

#### 2. 콘텐츠별 성과
- **UI Elements**:
  - 콘텐츠 목록 (캠페인, 날짜, 도달, 인게이지먼트)
  - 개별 성과 상세 (좋아요, 댓글, 저장, 공유)
  - 성과 트렌드
- **CTA**: "상세 보기"
- **Next**: 콘텐츠 상세

## Business Rules

### 자동 수집 데이터
- 노출수 (Impressions)
- 도달률 (Reach)
- 좋아요, 댓글, 저장, 공유
- 프로필 방문
- 웹사이트 클릭 (UTM 자동 생성)

### 데이터 수집 주기
- 콘텐츠 게시 후 24시간: 매 시간
- 24시간 이후: 매일 1회
- D+7 이후: 주 1회

### 리포트 자동 생성
- 일일 성과 요약: 매일 오전 9시 이메일 (옵션)
- 주간 리포트: 매주 월요일 PDF
- 캠페인 종료 리포트: 자동 생성

### ROI 계산
```
ROI = (전환 가치 - 캠페인 비용) / 캠페인 비용 × 100
CPE = 캠페인 비용 / 총 인게이지먼트
CPR = 캠페인 비용 / 총 도달
```

## Edge Cases
- 인스타그램 API 제한: 캐시된 데이터 표시, 실시간 갱신 지연 알림
- 콘텐츠 삭제됨: 마지막 수집 데이터로 표시, 삭제 표시
- 전환 추적 미연동: ROI 계산 제외, 연동 가이드 제공
