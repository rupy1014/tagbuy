# Spec: 002-campaign

## Feature Name
캠페인 생성 및 관리

## User Value (고객 가치)
- 간단한 입력으로 캠페인 빠르게 생성
- 명확한 타겟 조건 설정으로 적합한 인플루언서 매칭
- 실시간 캠페인 상태 및 성과 확인

## Target User (누가 쓸까?)
- Primary: 광고주 (중소기업, 스타트업, 마케팅 대행사)
- Use case: 인플루언서 마케팅 캠페인 런칭

## User Journey & Screen Flow

### 1. 캠페인 생성 - 기본 정보
- **UI Elements**: 캠페인명, 목표 선택 (브랜드 인지도/판매 전환/앱 설치 등), 기간 설정
- **CTA**: "다음"
- **Next**: 예산 설정 페이지

### 2. 캠페인 생성 - 예산 설정
- **UI Elements**: 총 예산 입력, PG 수수료 부담 주체 선택, 예상 비용 계산기
- **CTA**: "다음"
- **Next**: 타겟 조건 페이지

### 3. 캠페인 생성 - 타겟 조건
- **UI Elements**: 팔로워 수 범위, 카테고리 선택, 지역 선택, 인게이지먼트율 최소 기준
- **CTA**: "다음"
- **Next**: 가이드라인 작성 페이지

### 4. 캠페인 생성 - 가이드라인
- **UI Elements**: 제품/서비스 정보, 필수 해시태그, 필수 문구, 금지 사항, 참고 이미지 업로드
- **CTA**: "미리보기", "결제하기"
- **Next**: 결제 페이지 (003-payment)

### 5. 캠페인 목록
- **UI Elements**: 캠페인 카드 (상태, 예산, 진행률, 도달 수), 필터 (상태별)
- **CTA**: "새 캠페인 만들기", "상세보기"
- **Next**: 캠페인 상세 페이지

### 6. 캠페인 상세
- **UI Elements**: 성과 지표, 참여 인플루언서 목록, 콘텐츠 목록, 정산 현황
- **CTA**: "인플루언서 추가 모집", "캠페인 종료"
- **Next**: 인플루언서 상세, 콘텐츠 검수 (005-content)

## Business Rules
- 최소 예산: 50만원
- 캠페인 기간: 최소 3일 ~ 최대 90일
- 정산 규칙 선택: 표준(D+7), 단기(D+3), 장기(D+30), 즉시(D+1)
- 캠페인 상태: 준비중 → 진행중 → 검수중 → 정산중 → 완료
- 캠페인 수정: 결제 전까지만 가능

## Edge Cases
- 예산 소진 전 캠페인 종료: 잔여 예산 에스크로에서 환불
- 인플루언서 미선정: D+14까지 선정 없으면 전액 환불
- 캠페인 중도 취소: 진행 중인 건 정산 완료 후 잔액 환불

## Dependencies
- 003-payment (결제 및 에스크로)
- 004-matching (인플루언서 매칭)
- 005-content (콘텐츠 검수)
- 007-dashboard (성과 분석)
