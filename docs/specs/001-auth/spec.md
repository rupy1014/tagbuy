# Spec: 001-auth

## Feature Name
회원가입/로그인

## User Value (고객 가치)
- 광고주와 인플루언서 역할에 맞는 맞춤형 온보딩 경험
- 인스타그램 연동으로 인플루언서 프로필 자동 생성
- 간편하고 안전한 인증 프로세스

## Target User (누가 쓸까?)
- Primary: 광고주 (중소기업, 스타트업), 인플루언서 (마이크로, 나노)
- Use case: 서비스 최초 가입 및 로그인

## User Journey & Screen Flow

### 1. 랜딩 페이지
- **UI Elements**: 서비스 소개, 회원가입 CTA, 로그인 링크
- **CTA**: "광고주로 시작하기", "인플루언서로 시작하기"
- **Next**: 역할 선택 → 회원가입 페이지

### 2. 회원가입 (광고주)
- **UI Elements**: 이메일, 비밀번호, 기업명, 담당자명, 연락처
- **CTA**: "가입하기"
- **Next**: 이메일 인증 → 결제수단 등록 → 대시보드

### 3. 회원가입 (인플루언서)
- **UI Elements**: 이메일, 비밀번호, 인스타그램 연동 버튼
- **CTA**: "인스타그램으로 가입", "이메일로 가입"
- **Next**: 인스타그램 OAuth → 프로필 확인 → 대시보드

### 4. 로그인
- **UI Elements**: 이메일, 비밀번호, 소셜 로그인 버튼
- **CTA**: "로그인", "비밀번호 찾기"
- **Next**: 대시보드

## Business Rules
- 이메일 중복 가입 불가
- 인플루언서는 인스타그램 계정 연동 필수 (팔로워 수 확인용)
- 광고주는 기업 정보 입력 필수
- 비밀번호는 8자 이상, 영문+숫자+특수문자 조합

## Edge Cases
- 인스타그램 연동 실패 시: 수동 프로필 입력 허용 (관리자 승인 필요)
- 이메일 인증 만료: 재발송 요청 가능
- 소셜 계정 연결 해제: 이메일/비밀번호로 전환

## Dependencies
- NextAuth.js (인증 라이브러리)
- Instagram Graph API (인플루언서 계정 연동)
- 이메일 발송 서비스 (인증 메일)
