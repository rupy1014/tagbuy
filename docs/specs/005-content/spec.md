# Spec: 005-content

## Feature Name
콘텐츠 제출 및 검수

## User Value (고객 가치)
- AI 자동 검증으로 가이드라인 준수 여부 즉시 확인
- 광고주는 AI 리포트 참고하여 빠른 검수
- 인플루언서는 제출 전 자가 점검으로 승인률 향상
- 객관적 증거 기반 분쟁 예방

## Target User (누가 쓸까?)
- Primary (인플루언서): 콘텐츠 제작 및 제출
- Primary (광고주): 콘텐츠 검수 및 승인/거절

## User Journey & Screen Flow

### 인플루언서 플로우

#### 1. 콘텐츠 제작 가이드
- **UI Elements**: 캠페인 가이드라인, 필수 해시태그, 필수 문구, 금지 사항, 참고 이미지
- **CTA**: "확인했습니다", "질문하기"
- **Next**: 콘텐츠 업로드

#### 2. 콘텐츠 업로드
- **UI Elements**:
  - 플랫폼 선택 (인스타그램/블로그)
  - 콘텐츠 URL 입력
  - 스크린샷 자동 캡처 미리보기
- **CTA**: "제출하기"
- **Next**: AI 검증 (30초) → 결과 확인

#### 3. AI 검증 리포트 (인플루언서 뷰)
- **UI Elements**:
  - 가이드라인 준수도 (%)
  - 체크리스트 (해시태그, 브랜드명 언급, 제품 이미지, 필수 문구)
  - 예상 성과 (도달, 인게이지먼트)
  - 문제점 및 개선 제안
- **CTA**: "재업로드", "이대로 제출", "이의 제기"
- **Next**: 광고주 검수 대기

#### 4. 검수 결과 확인
- **UI Elements**: 승인/수정요청/거절 상태, 광고주 코멘트
- **CTA**: "수정 후 재제출" (수정요청 시)
- **Next**: 승인 → 정산 대기 / 수정요청 → 재업로드

### 광고주 플로우

#### 1. 검수 대기 목록
- **UI Elements**:
  - 검수 대기 콘텐츠 카드 (인플루언서, 제출 시간, AI 점수)
  - 필터 (캠페인별, AI 점수별)
- **CTA**: "검수하기", "일괄 승인"
- **Next**: 콘텐츠 상세 검수

#### 2. 콘텐츠 상세 검수
- **UI Elements**:
  - 콘텐츠 미리보기 (인스타그램 임베드)
  - AI 검증 리포트
    - 가이드라인 준수도
    - 체크리스트 결과
    - 예상 성과
  - 인플루언서 이의 제기 (있는 경우)
- **CTA**: "승인", "수정 요청", "거절"
- **Next**: 결과 전송 → 승인 시 정산 트리거

#### 3. 수정 요청
- **UI Elements**: 수정 사유 선택/입력, 구체적 피드백 작성
- **CTA**: "수정 요청 전송"
- **Next**: 인플루언서에게 알림

#### 4. 거절
- **UI Elements**: 거절 사유 선택/입력
- **CTA**: "거절 확정"
- **Next**: 인플루언서에게 알림, 에스크로 환불 처리

## Business Rules

### AI 검증 항목
- 필수 해시태그 포함 여부
- 브랜드명/제품명 언급 횟수
- 필수 문구 포함 여부
- 금지어 사용 여부
- 제품 이미지 포함 여부
- 콘텐츠 품질 (이미지 해상도, 텍스트 가독성)

### 검수 타임라인
- 광고주 검수 기한: 48시간
- 48시간 내 미검수 시: 자동 승인 옵션 (광고주 설정)
- 수정 요청 후 재제출 기한: 72시간

### 승인 조건
- AI 준수도 90% 이상 + 필수 항목 모두 충족 → 자동 승인 옵션
- AI 준수도 70% 미만 → 광고주 필수 검토

### 정산 트리거
- 승인 → D+7 정산 예약 (기본)
- 정산 규칙에 따라 D+3, D+30, D+1 가능

## Edge Cases
- 콘텐츠 삭제 감지: 승인 후에도 매일 크롤링, 삭제 시 정산 취소 및 광고주 알림
- URL 접근 불가: 재제출 요청
- AI 검증 실패: 수동 검수로 전환
- 분쟁 발생: AI 리포트 + 스크린샷이 증거 자료
