# Spec: 008-fraud-detection

## Feature Name
AI 사기 탐지

## User Value (고객 가치)
- 가짜 팔로워/인게이지먼트 자동 탐지로 광고비 보호
- 신뢰할 수 있는 인플루언서만 매칭
- 부정 거래 실시간 감지 및 차단
- 플랫폼 전체 신뢰도 향상

## Target User (누가 쓸까?)
- Primary: 광고주 (신뢰할 수 있는 인플루언서 확보)
- Secondary: 플랫폼 운영자 (부정 행위 관리)

## User Journey & Screen Flow

### 광고주 플로우

#### 1. 인플루언서 프로필 - 신뢰도 점수
- **UI Elements**:
  - AI 품질 점수 (100점 만점)
  - 가짜 팔로워 추정 비율
  - 인게이지먼트 진정성 점수
  - 위험 신호 (있는 경우)
- **CTA**: "상세 분석 보기"
- **Next**: 상세 분석 리포트

#### 2. 상세 분석 리포트
- **UI Elements**:
  - 팔로워 분석
    - 실제 팔로워 비율
    - 봇/가짜 계정 비율
    - 비활성 계정 비율
  - 인게이지먼트 분석
    - 댓글 품질 점수
    - 좋아요 패턴 분석
    - 의심스러운 행동 이력
  - 성장 패턴
    - 팔로워 증가 추이
    - 급격한 변동 이력
- **CTA**: "매칭 진행", "매칭 제외"
- **Next**: 매칭 결정

#### 3. 이상 알림
- **UI Elements**:
  - 알림 유형 (팔로워 급변, 의심 활동 등)
  - 대상 인플루언서
  - 상세 내용
  - 권장 조치
- **CTA**: "상세 보기", "무시", "캠페인 중단"
- **Next**: 조치 결정

### 플랫폼 운영자 플로우

#### 1. 사기 탐지 대시보드
- **UI Elements**:
  - 의심 계정 목록
  - 신고 접수 현황
  - 조치 필요 건
  - 트렌드 분석
- **CTA**: "계정 검토", "일괄 처리"
- **Next**: 계정 상세 검토

#### 2. 계정 검토 및 조치
- **UI Elements**:
  - 계정 상세 정보
  - AI 분석 결과
  - 과거 신고 이력
  - 조치 옵션 (경고/제한/정지/영구정지)
- **CTA**: "경고 발송", "계정 정지", "정상 처리"
- **Next**: 조치 완료

## Business Rules

### AI 검증 항목

#### 1. 팔로워 품질 분석
- 봇 계정 탐지 (프로필 사진 없음, 게시물 없음, 팔로워/팔로잉 비율 이상)
- 비활성 계정 비율
- 팔로워 지역 분포 (타겟과 불일치 시 의심)
- 팔로워 구매 패턴 (급격한 증가 후 감소)

#### 2. 인게이지먼트 진정성
- 댓글 품질 (스팸성 댓글, 이모지만 있는 댓글 비율)
- 좋아요 타이밍 (게시 직후 비정상적 급증)
- 리치 대비 인게이지먼트 비율
- 댓글 작성자 계정 품질

#### 3. 성장 패턴 분석
- 팔로워 증가 속도
- 급격한 팔로워 변동 (1일 10% 이상)
- 계정 연령 대비 성장률

### 위험 등급
| 등급 | 기준 | 조치 |
|------|------|------|
| 안전 (녹색) | 품질 점수 80점 이상 | 정상 매칭 |
| 주의 (노란색) | 품질 점수 60-79점 | 경고 표시, 광고주 판단 |
| 위험 (빨간색) | 품질 점수 60점 미만 | 매칭 제한, 검토 필요 |
| 차단 | 확인된 부정 행위 | 계정 정지 |

### 자동 조치
- 품질 점수 40점 미만: 자동 매칭 제외
- 팔로워 1일 20% 이상 급증: 자동 검토 대상
- 부정 행위 3회 신고: 자동 검토 대상
- 확인된 부정: 계정 즉시 정지

## Edge Cases
- 바이럴로 인한 급성장: 정상 성장과 구분 필요, 수동 검토
- 인플루언서 항의: 재검토 요청 프로세스
- AI 오탐: 수동 검토 후 정상 처리, AI 학습 데이터 업데이트
- 기존 광고주와 진행 중인 부정 계정: 광고주에게 알림, 캠페인 유지/중단 선택권

## Dependencies

- 004-matching (매칭 시스템)
- Instagram Graph API (계정 데이터)
- AI/ML 서비스 (TensorFlow)
- 신고 시스템
