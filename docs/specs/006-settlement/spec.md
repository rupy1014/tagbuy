# Spec: 006-settlement

## Feature Name
실시간 정산 시스템

## User Value (고객 가치)
- 승인 후 10분 내 실시간 정산 (업계 유일)
- 365일 24시간 언제든 출금 가능
- 정산 옵션 선택 (표준/즉시/예약)
- 투명한 정산 내역 및 세금 처리

## Target User (누가 쓸까?)
- Primary: 인플루언서 (정산 수령)
- Secondary: 광고주 (정산 현황 확인)

## User Journey & Screen Flow

### 인플루언서 플로우

#### 1. 정산 대시보드
- **UI Elements**:
  - 이번 달 수익
  - 정산 대기 금액
  - 출금 가능 금액
  - 월별 수익 그래프
- **CTA**: "즉시 출금하기", "정산 내역 보기"
- **Next**: 출금 화면 / 정산 내역

#### 2. 정산 옵션 선택
- **UI Elements**:
  - 표준 정산 (무료): 24시간 이내 자동 입금
  - 즉시 정산 (+0.5%): 10분 이내 입금
  - 예약 정산 (무료): 날짜/시간 지정
- **CTA**: "정산 방식 선택"
- **Next**: 정산 처리

#### 3. 출금 신청
- **UI Elements**:
  - 출금 가능 금액
  - 출금 금액 입력 (최소 1만원)
  - 출금 계좌 선택/등록
  - 예상 입금 시간
- **CTA**: "출금 신청"
- **Next**: 출금 처리 → 완료 알림

#### 4. 정산 내역
- **UI Elements**:
  - 정산 목록 (날짜, 캠페인, 금액, 상태)
  - 필터 (기간별, 캠페인별, 상태별)
  - 원천징수 내역 (3.3%)
- **CTA**: "상세 보기", "증빙 다운로드"
- **Next**: 상세 내역

#### 5. 계좌 관리
- **UI Elements**: 등록된 계좌 목록, 새 계좌 등록 폼
- **CTA**: "계좌 추가", "기본 계좌 설정"
- **Next**: 계좌 등록 완료

### 광고주 플로우

#### 1. 정산 현황 (캠페인별)
- **UI Elements**:
  - 총 집행 금액
  - 정산 완료 / 정산 예정 / 잔여 예산
  - 인플루언서별 정산 현황
- **CTA**: "상세 보기"
- **Next**: 정산 상세

## Business Rules

### 정산 옵션
| 옵션 | 수수료 | 처리 시간 | 설명 |
|------|--------|----------|------|
| 표준 정산 | 무료 | 24시간 이내 | 기본 정산 방식 |
| 즉시 정산 | 0.5% | 10분 이내 | 365일 24시간 가능 |
| 예약 정산 | 무료 | 지정 시간 | 원하는 날짜/시간 설정 |

### 정산 트리거 조건
1. 광고주 콘텐츠 승인 완료
2. 정산 규칙 기간 경과 (D+7 기본)
3. 콘텐츠 유지 확인 (삭제/비공개 없음)

### 자동 집행 플로우
1. 승인 → 정산 예약 (D+7)
2. D+1~D+7: 매일 콘텐츠 유지 확인
3. D+7: 조건 충족 시 에스크로 → 인플루언서 계좌
4. 정산 완료 알림

### 세금 처리
- 원천징수: 3.3% 자동 차감
- 월별 수익 리포트 제공
- 연말 종합소득세 신고 자료 제공

### 출금 규칙
- 최소 출금액: 1만원
- 출금 수수료: 무료
- 출금 한도: 일 1,000만원, 월 5,000만원

## Edge Cases
- 콘텐츠 중도 삭제: 정산 취소, 에스크로 광고주 환불
- 정산 실패 (계좌 오류): 재시도 3회 후 수동 처리, 고객센터 연결
- 분쟁 중 정산: 분쟁 해결 전까지 보류
- 계좌 미등록: 정산 보류, 계좌 등록 요청 알림
