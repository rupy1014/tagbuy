# TagBuy 서비스 컨셉

## 서비스 본질
에스크로 0.5% + PG 실비 투명 과금, 10분 만에 정산받는 가장 빠르고 투명한 인플루언서 마케팅 플랫폼

## 해결하는 문제

### 광고주의 문제
- 기존 플랫폼 수수료 3%로 비용 부담
- 외부 PG 리다이렉트로 결제 UX 불편, 이탈률 높음
- 수수료 내역 불투명 (에스크로 + PG 수수료 묶음)
- 정산 1-2주 소요로 인플루언서 확보 경쟁에서 불리

### 인플루언서의 문제
- 정산까지 1-2주 대기 (캐시플로우 문제)
- 주말/공휴일 정산 불가
- 가짜 인플루언서와의 경쟁으로 신뢰도 하락
- PG 수수료 부담 여부 선택권 없음

## 타겟 고객

### Primary Persona

**광고주**
- 중소기업 (연매출 10억-100억): 마케팅 예산 제한적, 저렴한 수수료와 명확한 ROI 필요
- 스타트업: 빠른 성장 추구, 효율적인 마케팅과 데이터 필요

**인플루언서**
- 마이크로 인플루언서 (1만-10만 팔로워): 높은 인게이지먼트, 빠른 정산과 투명한 거래 필요
- 나노 인플루언서 (1천-1만 팔로워): 니치 카테고리 강점, 접근성과 간편한 사용 필요

### Secondary Personas
- 마케팅 대행사
- 매크로 인플루언서 (10만+ 팔로워)
- MCN (멀티채널 네트워크)
- 크리에이터 에이전시

## 핵심 가치 제안

1. **투명한 수수료**: 에스크로 0.5% + PG 실비, 경쟁사 대비 최대 73% 절감
2. **실시간 정산**: 승인 후 10분 내 입금, 업계 유일
3. **원클릭 통합 UX**: 외부 리다이렉트 없이 5초 결제 완료
4. **PG 수수료 선택권**: 광고주가 부담 주체 선택 가능 (경쟁사 불가)
5. **AI 신뢰 시스템**: 가짜 팔로워 탐지, 콘텐츠 자동 검증

## 주요 기능 (고객 가치 중심)

- **001-auth**: 회원가입/로그인 (광고주/인플루언서 역할 분리)
- **002-campaign**: 캠페인 생성 및 관리 (목표, 예산, 타겟 조건 설정)
- **003-payment**: PG 결제 + 에스크로 자동 예치 (원클릭 5초 결제)
- **004-matching**: 인플루언서 매칭 (AI 추천 + 수동 검색)
- **005-content**: 콘텐츠 제출 및 검수 (AI 자동 검증 리포트)
- **006-settlement**: 실시간 정산 시스템 (10분 내 입금)
- **007-dashboard**: 성과 대시보드 (실시간 지표, ROI 분석)
- **008-fraud-detection**: AI 사기 탐지 (가짜 팔로워, 부정 거래)

## 비즈니스 모델

### 주수익
- 에스크로 수수료: 거래액의 0.5%

### 부수익
- 플랫폼 이용료: Basic(무료), Pro(월 5만원), Business(월 20만원)
- 즉시 정산 수수료: 정산액의 0.5%
- B2B API 이용료: 월 30만-100만원
